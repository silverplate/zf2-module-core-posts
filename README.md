Posts module for Zend Framework 2
=================================

To add module to your project do:

    cd `zf2 app folder`
    git submodule add https://github.com/silverplate/zf2-module-core-posts.git module/CorePosts

Depends on [CoreApplication](https://github.com/silverplate/zf2-module-core-application), [CoreControl](https://github.com/silverplate/zf2-module-core-control), [CoreGeo](https://github.com/silverplate/zf2-module-core-geo) and [CorePeople](https://github.com/silverplate/zf2-module-core-people) modules.
